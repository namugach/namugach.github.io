mlops docker github action 연결하기

작업 환경

windows: wsl2

mac도 비슷할 것임.

docker

Dockerfile

이동

cd

열기

vi Dockerfile

작성

# Ubuntu 기반의 Docker 이미지 생성
FROM ubuntu:24.04

# 필수 패키지 설치
RUN apt-get update && \
  apt-get install -y \
  docker.io \
  openssh-server \
  curl \
  net-tools \
  git \
  wget \
  tree \
  vim \
  && rm -rf /var/lib/apt/lists/*

# .bashrc 파일을 root 홈으로 복사
RUN cp /home/ubuntu/.bashrc /root/.bashrc


# Docker 데몬과 SSH 데몬을 동시에 실행
CMD ["/bin/bash", "-c", "dockerd"]

빌드

docker build -t ubuntu-dind:v1 .

컨테이너

중요!!

docker 컨테이너 안에서 docker를 사용할 때는

꼭

--privileged

옵션을 붙여서 컨테이너를 생성해야 한다.

생성 및 실행

docker container run --privileged -itd --name ml-pipe ubuntu-dind:v1

접속

docker container exec -it ml-pipe bash

확인

입력

docker container ls

출력

ONTAINER ID   IMAGE     COMMAND   CREATED   STATUS    PORTS     NAMES

Poetry

설치

curl -sSL https://install.python-poetry.org | python3 -

열기

vi ~/.bashrc

추가

export PATH="/root/.local/bin:$PATH"

적용

source .bashrc

확인

poetry --version

git remote -v

vi README.md

git commit -m "start"
