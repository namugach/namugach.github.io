문제

용량이 무려
173g
  • wsl은 자동으로 용량을 줄이지 않음
  • wsl에서 용량 확보를 해도, 그 안에 공기는 그대로
  • 때문에 공기 빼주는 작업을 수동으로 해야함

용량 확인

설치

sudo apt-get install ncdu

조회

sudo ncdu / --exclude /mnt
Info
  • --exclude: 제외 디렉토리

sudo를 넣지 않으면 권한이 없기 때문에 모든 디렉토리를 순회하지 않는다.

94.5 gb
