sudo apt-get update
sudo apt-get -y install mysql-server

이렇게 뜨고, shell이 멈춤
ctrl + c 로 캔슬로 안되고..

나갔다 들어오면 잘 되긴 함.

시작

sudo service mysql start

계정

mysql 계정 생성시 %의 의미

최조 접속

sudo mysql -u root

만들기

CREATE USER 'ubuntu'@'%' IDENTIFIED BY '1234';
계정이 존재 하면 아래의 것으로 비번을 초기화 함
ALTER USER 'ubuntu'@'%' IDENTIFIED BY '1234';

권한 부여

GRANT ALL PRIVILEGES ON *.* TO 'ubuntu'@'%' WITH GRANT OPTION;

변경 사항 적용

FLUSH PRIVILEGES;

나가기

exit

설정

권한

sudo chown -R mysql:mysql /var/run/mysqld
sudo chmod 755 /var/run/mysqld

외부 접속 허용

설정

  • 열기
sudo vi /etc/mysql/mysql.conf.d/mysqld.cnf
  • 편집
port                    = 3306
bind-address            = 0.0.0.0

적용

sudo service mysql restart

확인

  • 입력
sudo service --status-all
  • 출력
 [ - ]  dbus
 [ + ]  mysql
 [ - ]  procps
 [ - ]  ssh
 [ - ]  x11-common

테스트

mysql -u ubuntu -p -h 자신의_아이피_주소
  • ex
mysql -u ubuntu -p -h 172.17.0.3
Important

테스트를 할 때, 설정을 한, 본 서버에서 접속이 된다면
타 서버에서도 접속 가능.
