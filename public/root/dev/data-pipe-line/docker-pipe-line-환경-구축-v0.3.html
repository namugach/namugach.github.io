설치

저장소 복제

git clone https://github.com/namugach/pipeline_base

이동

cd pipeline_base

실행

./install

docker

컨테이너 진입

창을

2개 띄워서 나중에 모니터링 할 것도 미리서 준비 해놓자

docker container exec -it server1 bash -c "cd /root && bash"

카프카 클러스터 서버 시작

./run/kafka/start_cluster.sh

모니터링

./test/monitor.sh

데이터 삽입

./test/main.sh
