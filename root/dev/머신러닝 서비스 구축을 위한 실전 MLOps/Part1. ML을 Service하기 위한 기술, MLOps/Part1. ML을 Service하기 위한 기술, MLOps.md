## MLOps란

- 다른 환경에서 n 명의 사람과 협업
- MLOps = ML + DevOps
- 아직은 생소한 분야
- ML을 효율적으로 개발하고 성공적으로 서비스화하고 우할 때 필요한 모든 것을 다루는 분야
